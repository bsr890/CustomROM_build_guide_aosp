# Custom Rom Build #
This is the Custom Rom build guide for Redmi Note 11 (spes/spesn)

## Requirements ##
1. A cloud server like GCS, AWS, AZURE, etc.
2. Minimum 4-core CPU with 8 GB of RAM and 500 GB+ storage. Higher is better for reducing compiling time. Here mine is "Intel(R) Xeon(R) Platinum 8488C equipped with 16-core CPU, 32 GB of RAM, and 1 TB of high-speed server-grade SSD."
3. Device: Redmi Note 11
4. Codename: spes/spesn
5. Stable internet connection
6. OS Ubuntu 20.4 LTS or a newer version
